## Instructions:
1. Open `Products-Import-WITH-SUBFAMILY.xlsx`
2. For each product, look at the `familyCode` to find the family
3. Use this guide to find the appropriate `subfamilyCode` based on the product type
4. Fill in the `subfamilyCode` column
5. Save and use that file for import
